# LMS Frontend

### setup instruction
1. clone the project
```
        git clone: https://github.com/Sunnyraj8789/lms-frontend.git
```
2. Move into the directory
```
    cd lms-frontend-hn
```

3. install dependencies
```
    npm i
```



4. run the server
```
    npm run dev
```
